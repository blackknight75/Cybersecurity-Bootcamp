# Raspberry Pi Setup

### Master Node Setup

- Install OS with SSH
- config sd card files
- Install iptables if neccesary
- sudo iptables -F
- sudo update-alternatives --set iptables /usr/sbin/iptables-legacy
- sudo update-alternatives --set ip6tables /usr/sbin/ip6tables-legacy
- sudo reboot
- link to master node
